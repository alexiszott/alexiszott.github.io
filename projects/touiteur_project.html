Touitteur - 2e année du BUT

Touiteur est un projet au cours duquel nous devions recréer une version simplifié du réseau social Twitter ( "X" désormais). Pour cela, nous étions quatre personnes par projet.

Une base de données SQL était là pour stocker toutes les informations. Sur notre version on peut :

  • s'inscrire
  • se connecter
  • se déconnecter
  • envoyer des tweets
  • suivre un utilisateur
  • suivre un hashtag
  • avoir une page d'accueil personnalisée avec nos abonnements
  • liker ou disliker un tweet

Tout cela a été développé en utilisant PHP, HTML et CSS. L'application web devait être un minimum sécurisée, on a donc filtré tout ce qui venait des utilisateurs afin d'éviter les injections SQL et XSS.
